# Automated Recon Tool (ART) - Streamlined Network Reconnaissance Tool
Network reconnaissance tool, bringing together WHOIS, DNS, port scanning, and HTTP header analysis to one tool. 

# Project Overview
A tool for network reconnaissance featuring:
- WHOIS Lookup
- DNS Enumeration
- Open Port Scanning
- HTTP Header Analysis
- Summary Report Output with a single-command interface

# Timeline
- Weeks 1-2: Project setup and skeleton
- Weeks 3-4: WHOIS and DNS modules
- Weeks 5-6: Port scanner and HTTP header analysis
- Weeks 7-8: Report integration and user testing, bug fixes and final polish
